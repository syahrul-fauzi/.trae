User Manual — Smart Business Assistant (SBA)

Tujuan
- Membimbing pengguna menjalankan alur utama: Chat Streaming, Generate Dokumen, dan Tenant Switcher.

Prasyarat
- Akun aktif dengan peran yang sesuai
- Workspace/Tenant tersedia

Alur Chat Streaming
- Buka halaman Chat
- Ketik pertanyaan; kirim pesan
- Tinjau jawaban yang muncul secara streaming
- Jika ada error/tool-call gagal: ikuti prompt perbaikan (human‑in‑the‑loop)

Generate Dokumen
- Akses menu Dokumen
- Pilih template; isi data
- Kirim untuk generate; pantau status
- Unduh dokumen dari link hasil

Tenant Switcher
- Buka Switcher; pilih tenant
- Pastikan branding dan data sesuai
- Konfirmasi operasi lintas tenant jika diminta

Tips
- Gunakan kata kunci jelas untuk hasil lebih akurat
- Simpan template yang sering dipakai

Troubleshooting
- Tidak ada jawaban: periksa koneksi atau ulangi pertanyaan
- Dokumen gagal: cek input & coba lagi; hubungi admin jika berulang
