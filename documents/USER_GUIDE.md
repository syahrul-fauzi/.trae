# SBA (Smart Business Assistant) - User Guide

## Getting Started

### What is SBA?

SBA is your intelligent business assistant that helps you manage conversations, get insights, and streamline your business operations through AI-powered chat capabilities.

### First Steps

1. **Sign Up**: Create your account using your email address
2. **Complete Profile**: Add your business information and preferences
3. **Start Chatting**: Begin conversations with your AI assistant
4. **Explore Features**: Discover tools and capabilities as you use the platform

## Dashboard Overview

### Navigation

**Main Menu** (Left Sidebar):
- **Dashboard**: Your home screen with overview
- **Chat**: Access your conversations
- **Settings**: Manage your account and preferences
- **Help**: Get support and documentation

**Top Bar**:
- **Search**: Find conversations and content

**Recent Conversations**:

## Chat Features

### Starting a Conversation

1. **Click "New Chat"** button in the sidebar
2. **Type your message** in the input field
3. **Press Enter** or click **Send** to submit
4. **Wait for response** from your AI assistant

### Message Types

**User Messages**:
- Regular text messages
- Questions and requests
- Commands and instructions

**AI Responses**:
- Text responses with formatting
- Tool execution results
- Data analysis and insights

### Message Actions

**For Your Messages**:
- **Edit**: Click the edit button to modify your message
- **Delete**: Remove messages you no longer need
- **Copy**: Copy message content to clipboard

**For AI Responses**:
- **Copy**: Copy the response text
- **Regenerate**: Get a different response
- **Like/Dislike**: Provide feedback on helpfulness
- **Expand Tools**: View detailed tool execution

### Tool Calls

When the AI uses tools to help you, you'll see:

**Tool Indicators**:
- **Status**: Pending, executing, completed, or failed
- **Type**: Search, analysis, calculation, etc.
- **Details**: Click to expand and see parameters and results

**Common Tools**:
- **Search**: Find information online
- **Analysis**: Process data and provide insights
- **Calculation**: Perform mathematical operations
- **Data Processing**: Transform and analyze datasets

## Conversation Management

### Organizing Conversations

**Starring**:
- Click the star icon to mark important conversations
- Starred conversations appear at the top of your list
- Easy access to frequently used chats

**Renaming**:
- Click the conversation title to edit it
- Use descriptive names for easy identification
- Titles help you find conversations later

**Archiving**:
- Archive old conversations to keep your list clean
- Archived conversations are still accessible
- Restore archived conversations when needed

### Search and Filter

**Search Conversations**:
- Use the search bar at the top of the sidebar
- Search by conversation title or content
- Results update as you type

**Filter Options**:
- **All**: Show all conversations
- **Starred**: Show only starred conversations
- **Recent**: Show conversations from the last week
- **Archived**: Show archived conversations

## Advanced Features

### File Attachments

**Upload Files**:
1. Click the **paperclip** icon in the chat input
2. Select files from your device
3. Wait for upload to complete
4. Files are automatically processed and analyzed

**Supported File Types**:
- Documents (PDF, DOC, TXT)
- Spreadsheets (CSV, XLS)
- Images (JPG, PNG, GIF)
- Code files (JS, PY, JSON)

**File Processing**:
- Text extraction and analysis
- Data visualization
- Content summarization
- Insight generation

### Voice Input

**Recording Messages**:
1. Click the **microphone** icon
2. Speak your message clearly
3. Click **stop** when finished
4. Message is transcribed automatically

**Voice Commands**:
- Natural language processing
- Multi-language support
- Noise cancellation
- Real-time transcription

### Keyboard Shortcuts

**Navigation**:
- `Ctrl/Cmd + K`: Open search
- `Ctrl/Cmd + N`: New conversation
- `Ctrl/Cmd + /`: Show shortcuts

**Chat**:
- `Enter`: Send message
- `Shift + Enter`: New line
- `Ctrl/Cmd + Enter`: Send message (alternative)
- `Escape`: Clear input

**File Operations**:
- `Ctrl/Cmd + Shift + U`: Upload file
- `Ctrl/Cmd + Shift + R`: Start/stop recording
- `Ctrl/Cmd + V`: Paste (supports images)

## Troubleshooting

### Common Issues

**Connection Problems**:
- Check internet connection
- Refresh the page
- Clear browser cache
- Try different browser

**Message Issues**:
- Messages not sending: Check input field
- No AI response: Wait a moment, try again
- Slow responses: Check internet speed
- Error messages: Contact support

**File Upload Issues**:
- File too large: Reduce file size
- Unsupported format: Convert to supported format
- Upload failed: Check internet connection
- Processing error: Try different file
