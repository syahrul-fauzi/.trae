# SBA Boilerplate Struktur Modul

Struktur modular untuk microservices SBA:
- /core — logika bisnis inti (DDD)
- /modules — fitur terpisah (FSD)
- /config — manajemen environment dan konfigurasi
- /docs — dokumentasi teknis modul
- /tests — unit dan integration tests

Gunakan `Makefile` dan `Dockerfile` standar, serta `package.json` dengan scripts terstandar.
