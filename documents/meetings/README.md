# Catatan Rapat (Meetings)

- Struktur penamaan: `MEETING_NOTES_YYYY-MM-DD.md`
- Sertakan ringkasan keputusan, peserta, agenda, aksi tindak lanjut, dan lampiran.
- Gunakan template: `TEMPLATE_MEETING_NOTES.md`.

## Tautan Terkait
- Keputusan proyek: `../decisions/`
- Laporan proyek: `../laporan-proyek/`
