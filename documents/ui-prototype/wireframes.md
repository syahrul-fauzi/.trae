Wireframe Antarmuka — SBA-Agentic

Layar Utama
- Dashboard: ringkasan tenant, aktivitas agent, status workflow
- Workspace: daftar channel, dokumen, dan konteks
- Agent Chat: jendela percakapan, context panel, tools panel
- Workflow Builder: kanvas node, sidebar komponen, inspector
- Settings: manajemen peran, kebijakan, integrasi

Flow Navigasi
- Dashboard → Workspace → Agent Chat → Workflow Builder → Settings

Komponen UI
- Sidebar navigasi, breadcrumb, tab context, node editor

Aturan Interaksi
- Keyboard accessible, focus management stabil, autosave pada perubahan
