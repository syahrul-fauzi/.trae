# Dokumentasi Apps — SBA-Agentic

Dokumentasi mendalam untuk setiap aplikasi di monorepo SBA-Agentic. Gunakan README di subfolder berikut:

- `documents/apps/api/README.md` — Orchestrator & Unified API
- `documents/apps/app/README.md` — Agentic Web App (AG‑UI)
- `documents/apps/docs/README.md` — BaseHub Docs Site
- `documents/apps/web/README.md` — Shared Web Surface / Marketing
- `documents/apps/admin/README.md` — Admin Console (tenant, RBAC, billing, observability)
- `documents/apps/worker/README.md` — Background Workers (jobs, retry, logging, scaling)
- `documents/apps/orchestrator/README.md` — Workflow Orchestration (DAG, scheduler, coordination)

Setiap README menyertakan gambaran, arsitektur, konfigurasi, keamanan, observabilitas, pengujian, deployment, SLO, dan roadmap.
