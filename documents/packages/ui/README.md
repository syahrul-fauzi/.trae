# UI Package — Design System & Components

## Tujuan
- Menyediakan komponen UI konsisten untuk AG‑UI, Admin, dan Web dengan aksesibilitas dan performa.

## Fitur
- Komponen dasar (Button, Input, Modal, Table), tema, i18n.

## Arsitektur
- Atomic Design; folder `atoms`, `molecules`, `organisms`.

## Standar Koding
- A11y (ARIA), CSP aman, no inline scripts, error taxonomy UI.

## Pengujian
- Unit untuk komponen, snapshots, a11y lint.
